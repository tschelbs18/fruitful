About Fruitful

Fruitful was a Harvard CS50W project built in June 2020 by Ted Schelble. It's an absolutely free, open source tool that you can check out (or contribute to) on Github. Please try it out and let us know what you like or don't via a feedback/error report. Thank you!

How Fruitful Works

  1. Add some tasks to your to-do list.
  2. Add some custom rewards to motivate yourself.
  3. Earn points by completing your assigned tasks.
  4. Use the points to buy rewards.
  5. Check out the all-time or daily leaderboards to see how you stack up.
  6. Repeat steps 1-4 and have a fruitful day!

If you have any questions - check out the Frequently Asked Questions page.
